## 前言

Oracle 一键安装脚本，演示华为欧拉 openEuler 22.03 LTS SP3(aarch64) 一键安装 Oracle 19C ARM 单机（全程无需人工干预）。

>
> **⭐️ 更多教程参考：[Oracle 一键安装脚本实操合集，持续更新中！！！](https://www.modb.pro/db/1773583263184031744)**

## 前置准备

* 1、系统组安装好操作系统（支持最小化安装）
* 2、网络组配置好主机网络，通常只需要一个公网 IP 地址
* 3、DBA 创建软件目录：`mkdir /soft`
* 4、DBA 上传 Oracle 安装介质（基础包，补丁包）到 /soft 目录下
* 5、DBA 上传 Oracle 一键安装脚本到 /soft 目录下，授予脚本执行权限：`chmod +x OracleshellInstall`
* 6、DBA 挂载主机 ISO 镜像，这里只需要 mount 上即可（这个很简单，不了解的可以百度下）
* 7、根据脚本安装脚本以及实际情况，配置好脚本的安装参数，在 /soft 目录下执行一键安装即可。

## 安装命令

使用标准生产环境安装参数：

```
# 根据脚本 README 或者 -h 命令提示，编辑好一键安装命令，进入 /soft 目录执行安装：
[root@openEuler22-arm ~]# cd /soft/
[root@openEuler22-arm soft]# chmod +x OracleShellInstall 

./OracleShellInstall -lf enp0s5 `# 主机网卡名称`\
-n openEuler22-arm `# 主机名`\
-dd /dev/sdc `# DATA 磁盘盘符名称`\
-o lucifer `# 数据库名称`\
-dp 'Passw0rd#PST' `# sys/system 用户密码`\
-ds AL32UTF8 `# 数据库字符集`\
-ns AL16UTF16 `# 国家字符集`\
-redo 10 `# 在线重做日志大小（M）`\
-opd Y `# 是否优化数据库`
```
