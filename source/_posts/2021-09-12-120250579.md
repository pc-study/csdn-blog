**增加 Swap**

```
## 从根目录划一块空间给 Swap
dd if=/dev/zero of=/swapfile bs=1G count=1
mkswap /swapfile
swapon /swapfile
echo "/swapfile swap swap defaults 0 0" >>/etc/fstab

```

## 临时生效
sysctl -w vm.swappiness=10
sysctl -a | grep vm.swappiness
cat /proc/sys/vm/swappiness
