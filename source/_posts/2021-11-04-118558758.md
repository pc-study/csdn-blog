很多朋友工作学习中需要在 Windows 上安装 Linux 系统，最常用的就是使用 VMware Workstation 虚拟机。

**本文将讲解如何在 VMware Workstation 上安装 Linux 主机！**

### 2、下载 Linux 系统

## 二、开始安装

### 1、创建 Linux 主机

确认安装好 VMware Workstation 虚拟机后，打开虚拟机软件，开始创建 Linux 系统。

新建虚拟机：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图2](/images/c7d8a375aafb9a7b036fdb2915e0442a.webp)  
 选择典型：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图3](/images/b7beb7fac967ce2b945808fec2304770.webp)  
 选择稍后安装操作系统：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图4](/images/bbc13f5695f544f4602561dafbcdbe4d.webp)  
 选择 Linux，选择需要安装的版本：  
 修改虚拟机名称和位置：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图5](/images/fe04d4c86af59f63a5ef3df67f7e7fdc.webp)  
 修改虚拟机磁盘空间大小：  
 选自定义硬件：  
 网卡改为桥接模式，CD/DVD 选择已下载好的 Linux 镜像文件，本文选择 Linux 7.6 版本：  
 **📢 注意：** 如果需要修改CPU，内存，网卡数量，硬盘等，可以在上面自定义硬件中修改。

创建完成：  
 **至此，Linux 主机创建成功，接下来开始安装 Linux 系统！**

### 2、安装Linux主机

打开上面创建好的主机：  
 回车跳过等待：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图6](/images/dde32bef09b29868e6443b551db67cd8.webp)  
 检查镜像中，可以 ESC 跳过：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图7](/images/afbf19ba758777d7340af7636bba3d74.png)  
 选择语言，开始安装：  
 选择时区为上海：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图8](/images/720bb2f2f36fea3c345ee17dbf296b41.webp)  
 关闭 KDUMP：  
 选择图形化界面安装：  
 选择自定义分区：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图9](/images/1e4213d93cfd09325740d2eb5beefcda.webp)  
 选完之后，点击 Done：  
 添加/boot分区，默认 `2GiB` 即可：  
 添加 swap 分区，建议等于物理内存大小，可大于物理内存：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图10](/images/c221a460a36954f95b8a4665a6ab09a5.webp)  
 剩余磁盘空间全部划给根目录 / 即可：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图11](/images/06087103ae6d8765d93745f1784eb2d1.webp)  
 配置网络和主机名：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图12](/images/e176a4570387a1949da0d87368c33371.webp)修改主机名为 redhat7，可自定义，无需保持一致：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图13](/images/f1c89e5057a0699f9c61122052021fac.webp)  
 配置网络为固定 IP，IP 等信息根据本机实际网络情况填写：  
 点击开始安装：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图14](/images/05f17d1f606c7d0caf8c89b37f898d93.webp)  
 修改 root 用户密码：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图15](/images/945d0f8ae3806aec495511ccd8b6fa01.webp)  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图16](/images/f970e150d21ec52a4e01bb43f59831df.webp)  
 如果提示密码强度弱，点击两次 Done 即可通过：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图17](/images/e94f3ec51803eede668a8cee79320b84.webp)  
 等待安装结束，点击 Reboot 重启主机：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图18](/images/07024d6ed16e30dc2408ae19b48d12d6.webp)  
 勾选 License 声明：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图19](/images/3679006021313ec5d4b126b34fc69c55.webp)  
 点击完成配置：  
 配置时区，选择上海：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图20](/images/093230776793929b9ea7fd8ea1e029f1.webp)  
 创建用户 zhangsan：  
 ![实战篇：VMware Workstation 虚拟机安装 Linux 系统 - 图21](/images/a1bc284f45c229d665835808351958c5.webp)  
 这里需要强密码，否则无法通过：  
 **至此，Linux主机安装成功！**
