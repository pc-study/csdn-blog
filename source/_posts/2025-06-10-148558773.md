### 前言

对于很多没有接触过 Linux 操作系统的用户来说，系统的安装过程可能会感到陌生和复杂。为了让大家更好地理解和掌握 Linux 的安装过程，本文将以 Oracle Linux 8 为例，详细讲解安装过程中的每一步操作。需要注意的是，虽然本教程以 Oracle Linux 为例，但其他 Linux 发行版的安装步骤大同小异，掌握了这种安装方式，可以帮助你应对大部分 Linux 系统的安装。

本文将通过虚拟化环境演示整个安装过程，确保你可以在虚拟机上顺利安装和配置操作系统。

### 启动并进入引导界面

在完成虚拟化配置后，启动虚拟机并进入操作系统的引导界面。此时，虚拟机会从你准备好的安装介质（ISO 文件）中引导并显示安装界面。你需要等待一段时间，直到系统完成引导并进入安装界面。

![手把手教你安装 Linux 操作系统](/images/f566085ba5de5bfdac4ec9cb8854ef86.webp)

### 进入正式安装界面

引导完成后，操作系统会显示安装界面。此时，你将看到一个图形化的安装界面，要求你选择安装语言等基本设置。

![手把手教你安装 Linux 操作系统 - 图2](/images/63ce9b1adec49a446787452b530fd534.webp)

### 选择操作系统语言

在正式开始安装之前，系统会要求你选择操作系统的语言。虽然大多数操作系统支持多种语言，但为了安装和后续的操作方便，建议选择英文作为系统语言。这样，不仅安装过程中的所有提示和错误信息会以英文显示，同时也能避免在使用过程中因中文显示导致的兼容性问题。

![手把手教你安装 Linux 操作系统 - 图3](/images/26ecb9878e430d42008492c7aa253d3e.webp)

### 配置安装选项

在进入安装的配置界面后，你将可以设置一些与系统硬件、网络、时区等相关的参数。

#### 关闭 KDUMP

KDUMP 是 Linux 系统的内核崩溃转储机制，它允许系统在出现严重错误（如内核崩溃）时，保存系统内存内容以供故障排查使用。在许多环境中，尤其是虚拟化环境下，通常不需要启用 KDUMP。因此，建议选择关闭 KDUMP：

![手把手教你安装 Linux 操作系统 - 图4](/images/6081bd9d97cb1a80a9e9ef91218706be.webp)

#### 选择时区

选择适合你所在地区的时区。在国内用户的情况下，通常选择上海时区。时区设置会影响系统时间和文件时间戳，因此非常重要。

![手把手教你安装 Linux 操作系统 - 图5](/images/bb436f381dbbfc27b326953b59e34f44.webp)

### 系统磁盘分区

接下来，你需要对系统磁盘进行分区。在 Linux 系统中，磁盘分区的方式可以有很多种，但通常我们会选择以下几种常见的分区方式。

#### 自动分区配置

如果你不熟悉 Linux 的分区方式，可以选择自动分区。系统会根据磁盘大小和默认配置，自动为你创建分区。

![手把手教你安装 Linux 操作系统 - 图6](/images/b5aadafd71f96ad21faee14becb1c6ef.webp)

#### 自定义分区

但是，通常系统自动分配的空间以及分区可能与自己的需求不一致，所以需要根据自己的需求调整分区大小，你可以手动设置根目录（/）、Swap 分区、/home 分区等。

##### 配置 Swap 分区

Oracle 数据库对 Swap 分区有一定要求，建议根据系统的内存大小，配置适当大小的 Swap 分区。通常，Swap 分区大小是物理内存的两倍。

![手把手教你安装 Linux 操作系统 - 图7](/images/a241fa1a000d34c91dba2a3fc760589f.webp)

##### 分配剩余空间给根目录（/）

在配置完 Swap 分区后，你可以将剩余的磁盘空间分配给根目录（/）。根目录是 Linux 文件系统的核心，所有操作系统文件和用户数据都会存储在这里。

![手把手教你安装 Linux 操作系统 - 图8](/images/597c6a31fd462b638c938600e1455ab1.webp)

#### 确认格式化

完成分区后，系统会提示你确认是否格式化磁盘。格式化会清空磁盘上的所有数据，确保磁盘准备好用于安装操作系统。

![手把手教你安装 Linux 操作系统 - 图9](/images/4a5bcb57d019d2bdc05141a80d5da22e.webp)

### 配置主机名和网络

在系统安装过程中，你还需要配置主机名和网络。配置网络时，可以选择自动获取 IP 地址（DHCP），但为了避免动态 IP 地址变化，建议使用手动配置（manual）模式并设置固定 IP 地址。

#### 配置主机名

主机名是操作系统的网络标识，它有助于你在网络中识别系统。通常，主机名可以根据系统的用途和环境来命名。

![手把手教你安装 Linux 操作系统 - 图10](/images/3ba0b57bbe21bd4135c3b11e7c46c5e4.webp)

#### 配置网络

你可以选择配置静态 IP 地址（手动设置）或让系统通过 DHCP 自动分配 IP 地址。如果你需要确保 IP 地址在每次启动时保持不变，建议配置静态 IP 地址。

![手把手教你安装 Linux 操作系统 - 图11](/images/32504c874c88578d9bba6815813dc7af.webp)

![手把手教你安装 Linux 操作系统 - 图12](/images/39330095866c549db8a23769b8106b46.webp)

### 设置 Root 密码

Root 用户是系统的超级管理员，拥有系统上的所有权限。设置一个强密码是保证系统安全的关键。确保密码足够复杂，并且容易记住。

![手把手教你安装 Linux 操作系统 - 图13](/images/75e9a9dd6565e63dcf6f0f9d7e23afd9.webp)

### 开始安装

完成所有配置后，点击“开始安装”按钮，操作系统将开始复制文件和配置系统。安装过程可能需要一些时间，具体时间取决于系统性能和磁盘速度。

![手把手教你安装 Linux 操作系统 - 图14](/images/9b48efc0a9b9f9b562d947122e8d093b.webp)

### 接受许可协议

在安装过程中，你需要阅读并接受操作系统的许可协议。通常，软件许可协议中包含了使用条款和条件，确认你理解并同意这些条款后，才能继续安装。

![手把手教你安装 Linux 操作系统 - 图15](/images/25ff035c530bd9271f8bf4d5e0dc63d5.webp)

### 安装完成与系统重启

当系统安装完成后，它会提示你重启计算机。重启后，你将进入 Oracle Linux 系统的首次配置界面，完成最后的配置。此时，你可以登录系统并开始使用。

![手把手教你安装 Linux 操作系统 - 图16](/images/ed269d699b48c42b1645f3b9a7eefe38.webp)

安装成功后，你的 Linux 系统就准备好使用了！
