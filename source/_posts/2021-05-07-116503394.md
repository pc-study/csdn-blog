## 三、安装

**注意：可通过./OracleShellInstall.sh --help查看命令帮助**

![](/csdn-blog/images/6578583bb62b94982df330d86c6ee3e7.png)

**安装命令如下：**

```
1. cd /soft
2. ./OracleShellInstall.sh -i 10.211.55.100 `#Public ip`\
3. -n s11g `# hostname`\
4. -o s11g `# oraclesid`\
5. -op oracle `# oracle user password`\
6. -b /oracle/app `# install basedir`\
7. -s AL32UTF8 `# characterset`\
8. -opa 31537677 `# oracle psu number`


AI写代码bash
```
