**📢 注意：**

* stanby log 日志大小与 redo log 日志保持一致
* stanby log 数量： `standby logfile=(1+logfile组数)*thread=(1+3)*1=4` 组,需要加 4 组 standby logfile

```
ALTER DATABASE ADD STANDBY LOGFILE thread 1 
group 5 ('+DATA') SIZE 120M,

group 8 ('+DATA') SIZE 120M,
```

### 3、 RMAN DUPLICATE

rman 恢复备库：

```
rman target sys/oracle@orcl1 AUXILIARY sys/oracle@orcldg
 
run {
allocate channel prmy1 type disk;
allocate channel prmy2 type disk;
allocate auxiliary channel aux1 type disk;
allocate auxiliary channel aux2 type disk;
DUPLICATE TARGET DATABASE FOR STANDBY FROM ACTIVE DATABASE DORECOVER NOFILENAMECHECK;
}
```
