## 前言

使用脚本安装 Oracle 数据库是一件省时又省事的事情，接下来我演示一下整个过程，11GR2/12C/18C/19C/21C 通用。

## 安装前准备

本脚本最初仅用于安装 Oracle 数据库，不支持操作系统的安装，请提前安装好 Linux 操作系统。

| 主机版本 | 主机内存 | 磁盘空间 | 主机IP |
| --- | --- | --- | --- |
| rhel8.6 | 8G | 50G | 10.211.55.100 |

接下来我演示下如何使用脚本一键安装 19c 单机数据库，步骤所有版本通用。

## 上传介质

首先，使用 SSH 工具连接到刚安装好的 Linux 主机，创建一个存放安装介质的目录，比如：`/soft`，然后上传安装所需的介质：

```
[root@orcl soft]# du -sh *
2.9G	LINUX.X64_193000_db_home.zip ## 19c 数据库安装包，官网可直接下载
176K	OracleShellInstall.sh ## 一键安装脚本
1.5G	p33806152_190000_Linux-x86-64.zip ## 19c 最新 RU 补丁，MOS 上可下载
119M	p6880880_190000_Linux-x86-64.zip ## 19c 最新 OPatch 补丁，MOS 上可下载
11G	rhel-8.6-x86_64-dvd.iso ## 安装 Linux 系统的 ISO 镜像文件，也可在虚机直接挂载
276K	rlwrap-0.42.tar.gz ## 优化 sqlplus、rman 等工具的小插件
```

上传好以上安装介质我们就完成了第一步☝🏻。

## 挂载 ISO 镜像

由于安装 Oracle 数据库需要安装一些 rpm 的依赖包，需要使用 `yum` 命令来安装，这就需要我们挂载 ISO 镜像来配置本地 YUM 源：

```
mount -o loop /soft/rhel-8.6-x86_64-dvd.iso /mnt
```

这里只需要执行 mount 命令挂载即可，配置本地 YUM 源咱们脚本里会自动配置，挂载好之后，执行 `df -h` 看到如下输出即代表成功挂载：

```
[root@orcl soft]# df -h | grep mnt
/dev/loop0                   11G   11G     0 100% /mnt
```

至此，所有准备工作就做完了，是不是很简单。

## 执行一键安装

进入安装介质的目录 /soft，根据前面讲解的参数详解篇，咱们编辑好一键安装的命令，直接黏贴执行即可：

```
./OracleShellInstall.sh -i 10.211.55.100 `#Public ip`\
-n orcl `# hostname`\
-o orcl `# oraclesid`\
-b /u01/app `# install basedir`\
-s AL32UTF8 `# characterset`\
-ns AL16UTF16 `# national characterset`\
-opa 33806152 `# Oracle RU NUMBER`
```

记住 `-i IP` 一定要填写自己的主机IP地址，不明白这些参数含义的可以回顾一下。
