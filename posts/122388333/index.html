前言

Linux 系统现在用的人越来越多，大家必不可少的学习一些基础来作为备用知识，不能说精通的话，至少要入门吧！本课程主要让大家了解一些基础的入门知识，比如 Linux 系统的安装，基础的命令以及如何简单的使用 Linux 系统。

需要学习 Linux，首先当然是要有一台 Linux 主机，所以第一课就是 Linux 系统的安装，本课程以 Centos 7.9 作为例子！
